我们正在建造什么

千载一瞬是一个持续生长的项目。从三位古人开始，逐步扩展人物、功能与体验深度。 以下是我们已经完成、正在进行和未来规划的一切。

↓
已完成

v1.0 — 千载初现

首版已于 2025 年发布，包含核心互动传记体验的完整闭环。

✓

三位首发古人

  • 王希孟 — 十八岁天才画师，《千里江山图》
  • 王羲之 — 书圣，《兰亭序》
  • 颜真卿 — 忠烈书家，《祭侄文稿》

每位角色含 6 种结局、完整分支剧情树、过渡叙事桥段。

✓

核心互动体验

  • 分支抉择引擎：选项 → 过渡剧情 → 下一节点
  • 三维卷轴拂尘系统（Three.js WebGL）
  • 结局图鉴 + 抉择轴线回溯
  • 延伸阅读书单 + 图书馆馆藏索引
  • 键盘快捷键（1/2 选择，Esc 关闭）
✓

视觉与体验

  • 电影感开场（Ken Burns 推镜 + 逐行揭示）
  • 水墨留白 · 宋体大字编辑式排版
  • 自定义光标（墨-ring）
  • 滚动揭示动画 + 页面转场墨幕
  • 背景音乐（可选关闭）
✓

性能优化

  • WebP 图片格式全面转换
  • Three.js 按需动态加载
  • 背景音乐延迟缓冲（preload=none）
  • 字体 swap + 系统字体降级
  • GitHub Pages 静态托管 + CDN 加速
进行中

v2.0 — 千载万象

这些是更长远的愿景。取决于反馈和资源，优先级可能调整。

01

声音与氛围

为每个角色定制专属背景音乐；加入环境音效（雨声、研磨声、卷轴展开声）；关键剧情节点触发音效。

预计：2025 Q3–Q4
02

多语言支持

英文版本（English），让海外用户也能体验中国古人的互动传记。包含完整的剧情翻译与文化注释。

预计：2025 Q4
03

社区与分享

结局分享卡片（生成带画作+结局标题的精美图片）；匿名统计数据（"73% 的人在这里选择了……"）；评论区或留言板。

预计：2026 Q1
04

更多时代与人物

扩展至更多朝代：战国（屈原）、汉（司马迁）、唐（李白、杜甫）、宋（李清照、辛弃疾）、明（王阳明）、清（曹雪芹）……

持续进行中
05

AI 驱动的动态叙事

探索引入大语言模型，在保持史料准确性的前提下，为用户选择生成更个性化的过渡叙述与结局变体。

探索阶段
06

无障碍与可访问性

完整的键盘导航支持、屏幕阅读器适配、减少动效模式增强、高对比度主题。

持续改进
参与

千载一瞬的代码与内容均托管在 GitHub。如果你对历史有研究、擅长写作、或者想做前端开发——欢迎参与。

前端功能开发

改进三维渲染、添加新交互、优化移动端体验。技术栈：原生 JS + Three.js。

内容审校与建议

纠正史料错误、改进叙事逻辑、提出新结局创意。任何形式的反馈都很宝贵。
